# CONTRIBUTING

> 🛡️ **RULE #1**: We only record **what the law itself says**, linked to official source. **NO opinions, NO advice.**

1. Always start from `data/TEMPLATE.json`
2. **EVERY legal fact → official working URL**
3. Always mark: `in_force`, `repealed`, dates
4. Plain language ≠ change the meaning
5. ❌ **NEVER personalised legal advice**
6. ❌ No AI content without full human check against official text
7. Everything = **GNU AGPL‑3.0**

✅ ONLY USE THESE SOURCES:
- legislation.gov.uk
- caselaw.nationalarchives.gov.uk
- bailii.org
- echr.coe.int

📂 Path: `data/country/region/act‑year/sXX‑title.json`

All changes checked against **4 Golden Rules** (`data/SCHEMA.md`).
